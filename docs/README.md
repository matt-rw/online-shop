# E-commerce CMS Platform

## Overview
A backend application built with Django, Tailwind CSS, and Stripe that supports structured content management and secure payment processing.

## Technology
Python, Django, Wagtail CMS, Tailwind CSS, Stripe

## Quick Start

### For Deployment
1. Start with [DEPLOYMENT_CHECKLIST.md](DEPLOYMENT_CHECKLIST.md)
2. Follow [DEPLOYMENT.md](DEPLOYMENT.md) for detailed steps
3. Use [GODADDY_DNS_SETUP.md](GODADDY_DNS_SETUP.md) for DNS configuration

### For Development
1. Run `./dev.sh` to start the development server
2. Or use Docker: see [DOCKER_SETUP.md](DOCKER_SETUP.md)
